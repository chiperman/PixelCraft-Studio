PixelCraft Studio 是一个专业级的网页端像素画编辑器，基于 React 开发。它轻量且功能强大，提供了诸如“图片转像素”、“参考图层”以及多语言支持等高级功能。

### ✨ 主要功能

*   **基础工具**：铅笔、橡皮擦、油漆桶填充、吸管（取色器）。
*   **图层系统**：
    *   **绘画图层**：主要的创作区域。
    *   **参考图层**：上传图片垫在画布下方用于临摹，支持调节透明度。
*   **图片处理**：
    *   **图片转像素**：导入任意图片，根据当前网格大小自动将其转换为像素画。
*   **画布管理**：
    *   自定义网格尺寸（提供预设，或手动设置最大 128x128）。
    *   缩放和平移控制。
    *   网格线显示/隐藏切换。
*   **项目管理**：
    *   保存项目为 `.json` 文件以便日后编辑。
    *   导出作品为高清 PNG 图片。
    *   自动保存至浏览器本地存储（LocalStorage）。
*   **个性化设置**：
    *   **主题**：浅色模式、深色模式及跟随系统。
    *   **多语言**：支持 11 种语言（英语、简体中文、繁体中文、日语、韩语、法语、德语、西班牙语、葡萄牙语、俄语、阿拉伯语）。
    *   **调色板**：默认 25 色调色板 + 5 个可编辑的自定义色块（双击编辑）。

### ⌨️ 快捷键与导航

| 操作 | 控制方式 |
| :--- | :--- |
| **平移 / 移动画布** | 按住 `空格键` + 鼠标左键拖拽 (或直接使用鼠标中键) |
| **缩放** | `Ctrl` + 鼠标滚轮 |
| **撤销** | 界面按钮 (支持历史记录) |
| **重做** | 界面按钮 |

### 🛠️ 技术栈

*   **框架**: React 19
*   **样式**: Tailwind CSS
*   **图标**: Lucide React
*   **语言**: TypeScript
